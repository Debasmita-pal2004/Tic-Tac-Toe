🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**.  
This project demonstrates core frontend concepts like DOM manipulation, event handling, and conditional logic.

 ✨ Features
- Two-player mode (Player X vs Player O)
- Dynamic turn switching
- Instant win detection (rows, columns, diagonals)
- Draw detection
- “New Game” and “Reset” functionality
